\question To include graphics, put your image in the "Graphics" folder and then use the command
\begin{verbatim}
    \includegraphics[]{Graphics/IMAGE_NAME}
\end{verbatim}
For example, the following image was included using the command
\begin{verbatim}
    \includegraphics[scale = 0.5]{Graphics/example_graphic.jpg}
\end{verbatim}
\begin{center}
    \includegraphics[scale = 0.5]{Graphics/example_graphic.jpg}
\end{center}
